hide (-)Welcome to the Payment Data Translator

This system acts as a hub for payments, translating it for and transferring it to and from various systems. To get started, select an item from the menu. If you do not see any selection options, please submit a request to be granted permission.

  • Transaction Detail Report

    This allows you to search by and view all the imported transaction fields.

  • Item Tracking Report

    This allows you to search by export batches and view all the exported transaction fields and batch details.

  • Import Errors

    This report lists any errors that were encountered during importing of transactions.

  • Daily Activity Summery

    This lists a summary of activity by date. It can be further broken out by either vendor or import batch (run).

  • Batch Search

    View a list of batches and their associated transactions.

  • Ledger Activity

    View payment records with attached ledger activity.

  • Schedule Activity Log

    View a history of all activity from the PDT Scheduling system. Regenerate export files or XML for web service calls for a given batch for manual processing.

  • Activity Log

    Lists all logged activity taking place in the system.

  • Schedules

    View and control the currently running automated scheduled tasks for the Payment Data Translator
